Examples
========

See the `OP2-APPS <https://github.com/OP-DSL/OP2-APPS>`_ repository for the latest generated parallel code for each application.

Airfoil
-------

* The Airfoil mesh (720,000 cells) can be downloaded `here <https://op-dsl.github.io/docs/OP2/new_grid.dat>`__.
* The Airfoil HDF5 double-precision mesh (720,000 cells) can be downloaded `here <https://github.com/ZamanLantra/OP2-ci_meshes/raw/refs/heads/main/airfoil/new_grid_dp.h5>`__. After downloading, rename ``new_grid_dp.h5`` to ``new_grid.h5``.
* The Airfoil mesh can also be generated using MATLAB with the mesh generator in ``OP2-Common/apps/mesh_generators/naca0012.m``.
* An older version of the Airfoil implementation is archived as a PDF `here <https://op-dsl.github.io/docs/OP2/airfoil-doc.pdf>`__.

Aero
----

* The Aero mesh (720,000 cells) can be downloaded `here <https://github.com/ZamanLantra/OP2-ci_meshes/raw/refs/heads/main/aero/FE_grid.dat>`__.
* The Aero HDF5 mesh (720,000 cells) can be downloaded `here <https://github.com/ZamanLantra/OP2-ci_meshes/raw/refs/heads/main/aero/FE_grid.h5>`__.
* The Aero mesh can also be generated using MATLAB with the mesh generator in ``OP2-Common/apps/mesh_generators/naca_fem.m``.

MGCFD
-----

* The OP2-ported MGCFD code is available in the `MG-CFD-app-OP2 repository <https://github.com/warwick-hpsc/MG-CFD-app-OP2>`__.
* The MGCFD mesh files can be downloaded `here <https://warwick.ac.uk/fac/sci/dcs/research/systems/hpsc/software>`__.
 
Volna
-----

* The OP2-ported Volna code is available in the `Volna-OP2 repository <https://github.com/reguly/volna>`__.
